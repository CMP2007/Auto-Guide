Estilos en elementos HTML

Por sí mismos, los elementos HTML no se enfocan en brindarle una imagen de interfaz muy agradable a los usuarios; su enfoque únicamente es el de definir la estructura y los elementos que se implementarán en la página. Para los estilos, la herramienta que se utiliza es el CSS. Esta tecnología está enfocada únicamente en enriquecer la visualización de la página; esto lo hace a través de un archivo CSS en el cual se definen todas las características con las que se mostrarán los elementos HTML. Para esto, cada grupo de caracteres debe de vincularse con los respectivos elementos en los que estos se aplicarán; esto lo hace a través de tres formas de vinculación:

  • Relacionándose a través del tipo de elemento.
  • Relacionándose a través del atributo "id" del elemento.
  • Relacionándose a través de una clase, la cual es llamada por el elemento HTML con el atributo "class", el cual puede ser implementado en cualquier elemento HTML.
